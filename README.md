# 酸素ボンベ残量計算機 (Oxygen Calculator PWA)

人工呼吸器を使用している患者様の酸素ボンベ使用可能時間を、安全係数(0.8)を考慮して素早く計算できるPWA（Progressive Web App）です。臨床工学技士の方向けに、シンプルで分かりやすいUIを提供します。

## 主な機能
- **酸素残量の計算**: 容器の内容積(L)と圧力計の数値(MPa)から残量(L)を算出。
- **酸素消費量の計算**: 分時換気量と設定酸素濃度(FiO2)から消費量(L/分)を算出。
- **使用可能時間の算出**: 残量と消費量から、計算上の使用可能時間、および安全係数(0.8)を掛けた安全な使用可能時間(分)を表示。
- **PWA対応**: スマートフォンやタブレットにインストール可能で、オフラインでも動作します。

## 計算式
1. **酸素残量(L)** = 容器の内容積(L) × 圧力計の数値(MPa) × 10
2. **酸素消費量(L/分)** = (分時換気量 ＋ フロートリガー) × ((酸素濃度(%) - 21) / 79)
3. **計算上の使用可能時間(分)** = 酸素残量(L) / 酸素消費量(L/分)
4. **安全な使用可能時間(分)** = 計算上の使用可能時間(分) × 0.8

## インストール方法
- iOS (Safari) の場合: 共有メニューから「ホーム画面に追加」
- Android (Chrome) の場合: メニューから「アプリをインストール」または「ホーム画面に追加」
